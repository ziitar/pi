## Auto-detect Logic

The classification-router.ts implements the following auto-detect logic:

1. On startup, check ~/.pi/agents/agents-index.md for version marker
2. If file exists and parseAgentIndex() returns valid AgentPool with version > 0:
   - Use multi-agent classification/routing
3. Otherwise:
   - Fall back to single-agent behavior (pi-classifier/pi-orchestrator)
